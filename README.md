# ArangoModel

_An ODM for ArangoDB in Crystal_

## WARNING: MAJOR WIP



## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  arango_model:
    github: lodestone/arango_model
```

## Usage

```crystal
require "arango_model"
```
